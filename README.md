# Portfolio Website - Riko Laksono

Website portfolio pribadi yang dirancang untuk menampilkan profil, keahlian, dan proyek-proyek yang telah saya kerjakan di bidang [Sebutkan Bidang Kamu, misal: Hardware Engineering / IT Support].

## 🚀 Fitur Utama
- **Responsive Design:** Tampilan optimal di berbagai perangkat (Mobile, Tablet, Desktop).
- **Portfolio Gallery:** Menampilkan proyek hardware dan sistem (Turnstile gate, Vending machine, Kiosk, dll) dengan efek Glassmorphism.
- **Contact Form Terintegrasi:** Form kontak yang terhubung langsung ke Google Sheets menggunakan Google Apps Script.
- **Smooth Scroll:** Navigasi antar bagian halaman yang halus.

## 🛠️ Teknologi yang Digunakan
- **HTML5** & **CSS3** (Custom Styling & Grid System)
- **JavaScript** (Fetch API untuk integrasi form)
- **Boxicons** (Untuk ikonografi)
- **Google Apps Script** (Sebagai backend untuk database form)
